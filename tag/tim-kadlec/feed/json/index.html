Tim Kadlec on Thriving in Unpredictability:

\n

\nThe only thing I control is my server environment. That\u2019s it. Everything else is completely unpredictable.So when I\u2019m building something, and I want to make it robust\u2014to make it resilient and give it the best chance it has to reach across this complicated mess full of unpredictability\u2014I want to take advantage of the one thing I control by letting my server output something usable and as close to working as possible.\n

\n

While Tim was talking about the web, this is the same line of thinking that goes into native apps. Being able to control both sides dramatically changes your assumptions about what you can get away with.
