\nShardQuery is a PHP class which is intended to make working with a partitioned dataset easier.

\n
    \n
  • QueryBroadcast – Send queries to all shards in a parallel pipelineParallelPipelining – Runs parts of queries in parallel and combines results
    • \n
  • QueryRouting – Sends queries only to the shard containing the requested data.
    • \n
  • ConditionPushdown – Aggregation, joins and filtering are always performed at the shard level (in the pipeline) which distributes the workGearman Workers – PHP is not threaded. Gearman (Net_Gearman) is leveraged to create each pipeline process.\n
