Update: Hacker News user tdmackey pointed out that Facebook has a job opening for a “Software Engineer, ARM Server“, with a specific goal in mind:

\n

\nWe aim to evaluate further improving the efficiency of our web tier by porting HipHopVM to new server hardware platforms based on power-efficient ARM.\n

\n

It isn’t clear if they’ll end up deploying a large number of ARM servers to production, but they are definitely looking to take them out for a test drive.

\n
\n

The latest update from Facebook about the HHVM JIT ended with this interesting note, emphasis mine:

\n

\nIt\u2019s also been crucial in our efforts to get hhvm running on ARM processors by isolating and reducing the amount of architecture-specific code we need to reimplement. Watch for an upcoming post devoted to our ARM port for more details!\n

\n

Why the big push to get HHVM ( the PHP Hip Hop Virtual Machine ) running on ARM CPUs? That seems to imply at least a desire to test out ARM based servers in Facebook data centers.

\n

I went through two posts from James Hamilton – ARM V8 Architecture and AMD Announces Server Targeted ARM, to catch up on ARM server expectations. Using ARM CPUs in data center servers may very well be the future. The AMD ARM server announced in October 2012 was expected to be:

\n

\nThis will be 4-core, 64 bit, more than 2Ghz part that is expected to sample in 2013 and ship in volume in early 2014.\n

\n

The latest news from AMD I could find on this was from a September 2013 press release:

\n

\n\u201cHierofalcon\u201d is the first 64-bit ARM-based platform from AMD targeting embedded data center applications, communications infrastructure and industrial solutions. It will include up to eight ARM Cortex-A57 CPUs expected to run up to 2.0 GHz, and provides high-performance memory with two 64-bit DDR3/4 channels with error correction code (ECC) for high reliability applications. The highly integrated SoC includes 10 Gb KR Ethernet and PCI-Express Gen 3 for high-speed network connectivity, making it ideal for control plane applications. The \u201cHierofalcon\u201d series also provides enhanced security with support for ARM TrustZone\u00ae technology and a dedicated cryptographic security co-processor, aligning to the increased need for networked, secure systems. \u201cHierofalcon\u201d is expected to be sampling in the second quarter of 2014 with production in the second half of the year.\n

\n

Instead of “more than 2Ghz” we are looking at “up to 2Ghz” and the schedule has been pushed back. That said the over all trajectory appears to be the same. Enough of the right features will be in place for an ARM server to be a potentially viable option.

\n

If Facebook isn’t already testing ARM servers I expect that they will be during 2014.
