The new Sacramento State logo was announced today. Along with the new logo, the main website launched a new design.

\n

The response around the office has been mixed. Myself and a few others first thought of a torch, while others thought the two Ss looked too much like dollar signs. I certainly like it much more than the old seal style logo. I also prefer the new website to the old one. I don’t like how the new site requires Javascript for even basic functionality. In many places the underlines for linked text has been removed, but kept in others. In some cases there is nothing short of a mouse over to provide any indication that some chunk of text is a link.

\n

All of this means new business cards, new signs and more hassle for everyone while we try to integrate the new logo. I still like it better than the old one though.
