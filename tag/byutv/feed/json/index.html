Last year my sister Beth showed me Studio C, a sketch comedy show produced by BYUtv. They’ve got a few recurring characters like The Awkward Avoidance Viking:

\n

\n

I was looking for a back episode recently, so I hit Google searching for ‘Studio C’. The first result was the home page for the show, then I saw the URL – http://www.byutv.org/show/c68c4e4f-6322-4a23-8b8e-a5be75b70635/studio-c – that is one ugly URL!

\n

My guess is that ugly looking piece in the middle is a unique identifier for the show that is used to look up the information needed to build the resulting page. If you know someone who works on www.byutv.org please ask them to fix that so that it can do lookups on nice, clean URLs.

\n

A better one would be byutv.org/show/studio-c. Don’t really need need /show/ in there, and the dash in studio-c should be optional. Using byutv.org/studio-c would be even better, with redirects so that if someone uses /studioc/ they still end up at the right place.
