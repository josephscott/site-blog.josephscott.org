As a communication tool, Slack is ok. What is definitely not great about Slack is the Mac OS client.

\n

For the last few years I’ve tried hard to avoid Java apps on Mac OS. They tend to be slow to start, memory hogs, and sometimes CPU hogs too. Unfortunately I’ve still had to make a few exceptions to this, and I continue to keep an eye out for alternatives.

\n

Lately installing the Slack Mac OS client feels a lot like installing a Java app. It is slow to start up, and is often a memory and CPU hog.

\n

I’d really like to see a native Mac OS Slack app. Or at least a version that doesn’t leave me feeling like Java on the desktop all over again.
