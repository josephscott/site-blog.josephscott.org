I was testing PHP code with VLD and noticed something odd. If I left off the final closing PHP then there were fewer ops. Here is an example:

\n

[sourcecode lang=”php”]
\n<?php
\n$first_name = ‘Joseph’;
\n$last_name = ‘Scott’;
\n?>
\n[/sourcecode]

\n

\n

The same file, minus the closing PHP tag:

\n

[sourcecode lang=”php”]
\n<?php
\n$first_name = ‘Joseph’;

\n[/sourcecode]

\n

has only 5 ops:

\n
\nline     # *  op                operands\n-----------------------------------------\n   2     0  >   EXT_STMT                                                 \n         1      ASSIGN        !0, 'Joseph'\n   3     2      EXT_STMT                                                 \n         3      ASSIGN        !1, 'Scott'\n   4     4    > RETURN         1\n\n
\n
