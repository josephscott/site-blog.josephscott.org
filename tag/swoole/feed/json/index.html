Swoole is another effort to bring event-driven async features to PHP:

\n

\nSwoole is an event-driven, asynchronous & concurrent networking communication framework with higher performance written only in C for PHP.\n

\n

While I’m not sure about using this as an application server for PHP ( I have mixed feelings about PHP app servers in general ), I do like their TCP server example:

\n

[sourcecode lang=”php”]
\n$serv = new swoole_server("127.0.0.1", 9501);
\n$serv->set(array(
\n ‘worker_num’ => 8,
\n ‘daemonize’ => true,
\n));
\n$serv->on(‘connect’, function ($serv, $fd){
\n echo "Client:Connect.n";
\n});
\n$serv->on(‘receive’, function ($serv, $fd, $from_id, $data) {
\n $serv->send($fd, ‘Swoole: ‘.$data);
\n $serv->close($fd);
\n});
\n$serv->on(‘close’, function ($serv, $fd) {
\n echo "Client: Close.n";
\n});
\n$serv->start();
\n[/sourcecode]
