Christian Zander sent an email to the FreeBSD-Hackers list detailing NVIDIA FreeBSD kernel feature requests. Having high quality video drivers is a very good thing:

\n

\nNVIDIA has been looking at ways to improve its graphics driver for the
\nFreeBSD i386 platform, as well as investigating the possibility of adding
\nsupport for the FreeBSD amd64 platform, and identified a number of
\nobstacles. Some progress has been made to resolve them, and NVIDIA would
\nlike to summarize the current status. We would also like to thank John
\nBaldwin and Doug Rabson for their valuable help.

\n

This summary makes an attempt to describe the kernel interfaces needed by
\nthe NVIDIA FreeBSD i386 graphics driver to achieve feature parity with
\nthe Linux/Solaris graphics drivers, and/or required to make support for
\nthe FreeBSD amd64 platform feasible. It also describes some of the
\ntechnical difficulties encountered by NVIDIA during the FreeBSD i386
\ngraphics driver’s development, how these problems have been worked around
\nand what could be done to solve them better.\n

\n

UPDATE Thu 13 Jul 2006 @ 2:30pm : BSDTalk has an interview with Andy Ritger and Christian Zander from nVidia.
