Matt made it so that the revision number shown in SVN-Notify links to the Trac diff page for that revision. This was for the wp-svn list, but no reason it couldn’t be used elsewhere.

\n

Figured I might need something like this one day, so here it is:

\n

\n
\n/usr/local/bin/svnnotify -r $REV -C -d -H HTML::ColorDiff -p $REPOS \n-t wp-svn@lists.automattic.com \n--from m@wordpress.org \n--reply-to wp-hackers@lists.automattic.com \n--revision-url http://trac.wordpress.org/changeset/%s \n--ticket-map '(#(d+))=http://trac.wordpress.org/ticket/%s' \n--ticket-regex '([(d+)])' \n--ticket-url http://trac.wordpress.org/changeset/%s\n
