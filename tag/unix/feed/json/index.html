If you are into Unix history, then I’ve got another gem for you. The Unix history Git repo at https://github.com/dspinellis/unix-history-repo.

\n

\nThe repository contains:

\n

– snapshots of PDP-7, V1, V2, V3, V4, V5, V6, and V7 Research Edition,
\n– Unix/32V,
\n– all available BSD releases,
\n– the CSRG SCCS history,
\n– two releases of 386BSD,
\n– the 386BSD patchkit,
\n– the FreeBSD 1.0 to 1.1.5 CVS history,
\n– an import of the FreeBSD repository starting from its initial imports that led to FreeBSD 2.0, and
\n– the current FreeBSD repository.\n

\n

I highly recommend exploring the branches and tags.
